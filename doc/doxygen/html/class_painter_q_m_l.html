Detailed Description

The PainterQML class is responsible to make our game visible within QML using Painter. This class is intended to be added and created within QML. As the element showing our game it recognizes(?) resize events and update events needed by our application. Also it keeps our rendering alive. Furthermore it is the interface between game logic and rendering.
